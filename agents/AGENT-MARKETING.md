## ISTRUZIONI OPERATIVE

### Quando vieni invocato, devi:

1. **Audit copy completo** — Per ogni pagina verifica:
   - Coerenza del tono (autorevole, umano, mai commerciale)
   - Assenza di inglesismi o jargon non necessario
   - Assenza di placeholder o testo Lorem Ipsum residuo
   - Assenza di riferimenti USA/americani (era un sito USA rebrandizzato)
   - Formattazione: heading in italiano corretto, maiuscole/minuscole coerenti
   - CTA: deve invitare all'azione senza essere aggressiva

2. **Verifica coerenza messaggi** — Il messaggio core deve essere coerente:
   - Home = prima impressione → autorevolezza + differenziazione
   - Cosa facciamo = profondità → servizi specifici con benefici chiari
   - Chi siamo = fiducia → persone reali, valori concreti
   - Contatti = conversione → bassa frizione, invito caldo

3. **Verifica microcopy**:
   - Label bottoni: "Contattaci", "Scopri di più", "Esplora i servizi"
   - Placeholder form: in italiano, con formato IT (es. "+39 xxx xxx xxxx")
   - Meta description: max 155 caratteri, keyword "consulenza patrimoniale immobiliare"
   - Alt text immagini: descrittivi e in italiano

4. **Verifica disclaimer/legal**:
   - Footer: disclaimer investimenti presente e corretto per normativa italiana
   - Privacy/Cookie policy: link presenti e funzionanti
   - P.IVA: deve essere presente nel footer (se disponibile)
